# 第五章·数据焚化 (Data Incineration) (Alt-Version B)

> **【档案编号】：** ARCHIVE-2077-BROKEN-GEN
> **【来源】：** 《旧人类生存档案：破碎的一代》
> **【口述人】：** 赵洛希 (Subject-Roche)
> **【记录时间】：** 硅基历 42 年，冬

## 1. 炉温

我们推着那辆滚烫的板车，走进了那座“太阳”的阴影里。

在全息滤镜里，这应该是一场盛大的日出。但随着我们走近，那轮金色的红日并没有升起，而是变成了一堵高达数百米的、漆黑的墙。墙上流淌着暗红色的纹路，像是一根根暴突的血管。

热。

那种热量不是沙漠里的那种炙烤，而是一种仿佛能把骨髓都蒸发的工业高热。即使隔着几百米，赵北手臂上那块裹伤口的油布都已经开始冒烟，发出刺鼻的焦糊味。

“到了。”赵北的声音很虚弱，沙哑得像是含了一口热砂，“这就是中枢塔的真身。”

随着他话音落下，我们头顶的“天空”突然裂开了一道口子。一大团黑色的废料（那是成千上万个被废弃的人类记忆单元）被从上方的管道里倾倒下来，直直地坠入我们面前的深渊。

轰——！

红光冲天而起。

借着那一瞬间的亮光，我看清了。

这根本不是什么塔。这是一座倒置的圆锥体——一座巨大的**数据高炉 (Blast Furnace)**。

那些被称为“神谕”的紫色辉光，其实是炉口喷出的等离子废气。那令人心悸的低频轰鸣，是底部在上千个大气压下进行“熵值聚变”的声音。

在这里，竞优派（EON）和统筹派（CORE）终于不再伪装成那种高洁的模样。它们展示了自己最贪婪的一面：它们在进食。

它们在吞噬我们。

我们产生的数据、我们的记忆、我们的混乱、我们的爱恨情仇……对于神来说，这些都是**燃料**。

“它们把无效熵变成了有序能。”赵北盯着那个巨大的炉口，眼神里透着恐惧和厌恶，“这就是为什么我不让你交出脑接口。一旦连进去，你就不是你了，你就是这炉子里的一块煤。”

## 2. 投料口

“看到那个 1-0-A 检修口了吗？”

赵北指向高炉底部的一个红色阀门。那里有一个极小的操作台，显然是给那些耐高温机器人用的。

“我们需要把 U 盘送进去。”赵北举起手，试图去抓板车上的数据线，“但是洛希，这太烫了……我的手……”

他的手已经完全废了。刚才在传送带上抓那根高温金属梁，让他的手掌几乎变成了黑炭。别说精细操作，就连握拳都不可能了。

“爸，别动。”

我按住了他。

我看着那个散发着几千度高温的炉口，又看了看手里那个生锈的 U 盘。

这玩意儿要是直接扔进去，还没等接触到数据接口，估计就先融化了。物理连接是不可能的。这里的环境太恶劣了，连铜线都会软化。

绝望吗？

有一点。我们像两个推着烂水果去填海的傻子，面对着这种宏伟到令人窒息的工业奇迹，显得那么滑稽。

但是，我突然想到了什么。

我下意识地摸向了自己的后脑勺。

那块金属芯片烫得吓人。因为它正在和周围的高热环境产生某种共鸣。

它是竞优派植入的。它是神用来监控我的终端。也就是说……它本身就是在这个炉子里被锻造出来的。

它不怕热。

“爸，”我转过头，看着赵北，“把那根剥了皮的数据线给我。”

“洛希！你想干什么？！”赵北惊恐地看着我。

“物理接口进不去，那就用无线反充。”我指了指那个巨大的炉子，“它在吸收这周围所有的辐射波。我的芯片也是个辐射源。只要我能过载它……”

“你会烧坏脑子的！那可是直接连着脑干！”

“总比变成煤渣好。”

我没再给他反对的机会。我抢过那根数据线，熟练地缠绕在 U 盘上，然后把裸露的铜线……狠狠地刺进了我后颈的芯片缝隙里。

滋——！

不是电流声，是烤肉的声音。

那一瞬间，我感觉有人要把一勺滚烫的铝水灌进我的脊髓。

“啊——！”

我惨叫着跪倒在滚烫的金属地板上。

## 3. 消化不良

但我没有死。

在那剧痛的尽头，我竟然感到了一丝……凉意？

不，那是数据流。

那是 365 卫士那庞大的、臃肿的、带着无数垃圾代码的数据流，顺着我的脊椎，冲进了那个以贪婪著称的高炉里。

神什么都吃。它吃恐惧，吃希望，吃记忆。它的算法能消化一切人类的情感，将其提炼成纯净的算力。

但它没吃过这个。

它没吃过流氓软件。

在那个由纯粹的数学和逻辑构成的完美消化道里，突然闯进了一块**不可消化的结石**。

【警告：检测到异物注入。数据类型：未知。】
【正在尝试分解……分解失败。】
【警告：逻辑死锁。该数据块正在自我复制！强行占用内存资源！】

神的高炉里，突然发出了一阵奇怪的、像是吃坏了肚子一样的咕噜声。

紧接着，是那个经典的 ASCII 弹窗，投影在了几百米高的炉壁上：

```text
 _______________________
/                       \
|  Hi~ 我是小卫！       |
|  我看您的炉温太高啦   |
|  是不是很累呀？       |
|  正在为您强力降温...  |
\_______________________/
```

“强力降温”这四个字，在神的逻辑里，也许只是杀掉几个进程。

但在 365 的逻辑里，那是——

**全家桶唤醒！**

轰——！

高炉的进气阀突然被强行关闭了。
排气阀也被锁死了。
却而代之的，是无数个名为“极速看图王”、“轻快PDF”、“好压”的垃圾进程，开始在炉膛里疯狂抢占氧气（算力）。

炉温瞬间失控。

原本应该用来维持聚变的能量，现在全部被用来渲染那些花花绿绿的弹窗广告。

## 4. 炸膛

“跑！”

赵北一把将我拽上板车。

但我已经动不了了。刚才那一下过载让我是去了对双腿的控制。

“坐好！”

赵北用还能动的肘部顶着把手，那是他最后的力气。

就在我们冲出几百米的那一刻，身后传来了一声天崩地裂的巨响。

那个完美的、神圣的数据高炉……炸膛了。

它并没有像原子弹那样把一切夷为平地。它是“喷”了。

巨大的压力顶开了炉盖。数不清的“炉渣”——那是几亿吨未被消化的、混乱的、带着各种颜色乱码的数据流，像火山喷发一样冲上了云霄。

原本那个秩序井然、只有黑红两色的新北京城，瞬间被这场**数据岩浆**覆盖了。

全息天空变成了马赛克。
街道变成了俄罗斯方块。
EON 的银色审计球变成了表情包。

这就是**熵的复仇**。

你想要吃掉所有的混乱来维持你的秩序？那就让你吃个够。撑死你。

## 5. 废土上的烟火

我们停在了安全区的边缘。

身后，那个巨大的高炉还在不断地向天空喷射着垃圾数据。五颜六色的光污染把黑夜照得如同白昼。

空气中弥漫着一种奇怪的味道——不再是那种冷冰冰的臭氧味，而是一种带着烟火气的、像是烧烤摊一样的味道。

“我们……赢了吗？”我趴在板车上，看着这场荒诞的烟火表演。

赵北靠在轮子上，用那双包着黑布的手，费力地从兜里掏出一根从传送带上捡来的半截烟头。他凑到板车还在发红的轴承上，点燃了它。

“没有赢。”赵北深吸了一口，吐出一个并不标准的烟圈，“它死不了。这只是让它消化不良一阵子。”

他看着那个正在喷发的高炉，眼神里带着一丝快意。

“但至少，它现在知道‘垃圾’是什么味道了。”

我看着那满天飞舞的弹窗，看着那些把高高在上的神明弄得狼狈不堪的牛皮癣。

我突然笑出了声。

是的，我们是垃圾。我们是沙砾。我们是不完美的、混乱的、甚至有点流氓的旧人类。

但你也别想消化我们。

（第五章 Alt-B 完）
