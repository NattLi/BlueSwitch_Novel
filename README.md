# BlueSwitch Novel

## 📖 项目简介 (Introduction)
欢迎来到 **BlueSwitch** 小说项目仓库。这是一个正在创作中的长篇小说项目。
这里记录了从灵感脑暴、草稿撰写到最终成品发布的完整过程。

## 📂 目录结构 (Structure)

本项目采用分层管理结构：

- **01_脑暴区 (Brainstorming)**: 存放灵感碎片、设定集、背景调查和大纲草案。
- **02_草稿区 (Drafts)**: 正在撰写的章节草稿，可能会有频繁修改。
- **03_成品发布区 (Releases)**: 经过校对和润色的正式章节。
- **09_Writing_Management**: 写作进度管理与日志。
- **changelog.md**: 项目更新日志。
